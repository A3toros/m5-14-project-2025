Create your own website using chat GPT

This website will show you how easy it is to create your own website using chat GPT. The key is to give it the detailed prompt, explaining exactly what you want it to do for you.

Different website design options

Prompt template

Write html code for simple 1 page website.
Header  _____ (write what content you want here)
Main part_____ (write what content you want here, if you have photos, write the names of the photo files, e.g. Photo1.jpg)
Footer  - “Mathayomwatsing school, 2025. All rights reserved”

CSS (styling):
Header – _____ (describe color you want) padding 20px top and bottom
Main part – _____ (describe color you want) padding left and right 5%, add red lines and indentation between lines
Footer – _____ (describe color you want) padding 15px top and bottom
Whole page: sans-serif font. Headers should be size 26, centered. Regular text – 18, justified alignment. Do not ever use <pre>

Add style tag for css, for computers and for mobiles.
